About the project

Title : Afalagi

It is a MERN fullstack web application, which registers freelancers and companies. React js , Bootstrap , CSS are used for the frontend and nodejs (& express) , and mongodb are used for the backend. The system allows registered companies to post a job , see who has applied for these jobs , close jobs , accept applications, sending connection requests and more. In the other hand freelancers can automatically see relevant jobs , apply to jobs , search jobs , edit their profile and more. It also includes security features like authentication and authorization.

Title : Tic tac toe

A simple Real time fullstack game application. The application uses socket.io to allow users play tic tac toe game using different browser (or devices) and see their stats. It automatically notifies to the other player when one of the player left the game. html , css and javascript are used for the frontend and nodejs for the backend.
